## 1. 性能需求

### 1.1 延迟 SLO

| 指标 | 目标值 | 度量方式 | 来源 |
|------|--------|----------|------|
| **问答端点 P95 延迟** | < **5000ms** | `response_time_ms` 字段 | `03` §4 |
| **研报对比查询响应** | ≤ **2000ms** | 服务端计时 | `03` §4 · `04` §4 |
| 会话管理端点延迟 | < 500ms | 服务端计时 | JSON I/O < 10ms |
| 前端页面首屏加载 | < 2000ms | 首屏 LCP | Vite 构建 + React SPA |
| Demo 模式响应时间 | ≤ **500ms** | `response_time_ms` | `04` §6 降级兜底可用 |

### 1.2 降级策略

**示例 — 三级降级链**：

```
CoPawAgent.ask(query, session_id)
  ├─[1] CoPaw 桥接 → answer_source='copaw', llm_used=True
  ├─[2] 百炼 DashScope → answer_source='bailian', llm_used=True
  └─[3] 离线演示兜底 → answer_source='demo', llm_used=False
```

| 约束 | 说明 |
|------|------|
| 三级降级**不可跳级** | 必须按 CoPaw → 百炼 → Demo 顺序尝试 |
| 降级**静默执行** | 不抛异常，通过 `answer_source` 标识 |
| Demo 模式**始终可用** | 无外部依赖，纯字符串拼接 |

## 2. 可用性需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 服务可用性 | **99%**（教学环境） | 单实例 Flask，不设 HA |
| MTTR | < 5 分钟 | 重启 Flask 进程即可恢复 |
| 故障模式与恢复 | LLM 提供商宏机→自动降级；Flask 进程崩溃→重启恢复 | 降级对用户透明，通过 answer_source 标识 |

## 3. 可观测性需求

### 3.1 链路追踪

| 项 | 规格 |
|----|------|
| traceId 格式 | `tr_{uuid.hex}`（32 位十六进制） |
| 注入位置 | 所有成功响应顶层 + 错误响应 `error.traceId` |
| 上游透传 | 优先复用请求头 `X-Trace-Id`，缺省时本地生成 |

### 3.2 错误码体系

| HTTP | `error.code` | 触发条件 |
|------|--------------|----------|
| 400 | `EMPTY_QUERY` | query 为空/空白/null |
| 400 | `INVALID_QUERY` | query >500 字符 或 缺少 session_id |
| 400 | `SESSION_NOT_FOUND` | 指定 session_id 不存在或已删除 |
| 400 | `INVALID_SESSION_ID` | session_id 不符合 UUID v4 格式 |
| 400 | `INVALID_FILE_FORMAT` | 研报文件格式非 pdf/html |
| 500 | `INTERNAL_ERROR` | Agent 编排内部异常 |

## 4. 安全需求

| 项 | 当前实现 | 说明 |
|----|----------|------|
| API 认证 | 无（开放访问） | Workshop 教学版 |
| CORS | 全开放 | 生产需白名单 |
| 速率限制 | 无 | 教学场景不限速 |

### 4.1 敏感数据保护

| 数据类型 | 保护方式 |
|----------|----------|
| LLM API Key | 环境变量存储（`.env`），禁止硬编码 |
| 用户查询 | 仅存储在本地 JSON，不传输到第三方（LLM 除外） |
| 研报文件 | 本地加密存储，禁止传输至外部服务（LLM 本地部署或内网百炼） |
| 系统日志 | 禁止记录完整 query/answer 内容，仅记录 traceId 和摘要信息 |

## 5. 数据约束

### 5.1 输入约束

| 约束 | 规格 | 来源 |
|------|------|------|
| `query` 最小长度 | 1 字符 | `04` R-02 |
| `query` 最大长度 | 500 字符 | `04` R-02 |
| `session_id` 必填 | 非空字符串 | `04` R-01 |
| `title` 默认值 | `"新会话"` | 09 §4 POST /sessions |
| 导出链接有效期 | 5 分钟 | `04` R-07 |
| 导出格式 | JSON 或 TXT | `04` R-07 |

### 5.2 类型约束

| 字段 | 类型 | 约束 |
|------|------|------|
| `llm_used` | `boolean` | 必须为 True/False，不得为字符串 |
| `answer_source` | `string` | 枚举：copaw \| bailian \| demo |
| `response_time_ms` | `integer` | 必须为非负整数，单位毫秒 |
| `model` | `string\|null` | Demo 模式时为 null，其余为模型标识字符串 |
| `session_id` | `string` | UUID v4 格式，36 字符（含连字符） |
